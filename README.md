# UnityPubSubSample

Samples of Pub/Sub Messaging for Unity.
The following samples are included.

- MessagePipe.VContainer
- MessagePipe.Zenject
- UniRx.MessageBroker
- Zenject.Signals

## Dependencies

* Unity 2020.3.5f1
